# 成績寫入 Google 雲端試算表

練習網頁**不能**像 Excel 那樣直接改試算表，必須透過你試算表裡的 **Apps Script** 幫你「加一列」。  
設定一次後，每次測完會自動寫入工作表 **`成績`**；開啟網站會寫入 **`造訪`**（同一瀏覽器每天一筆）。

你的試算表：  
https://docs.google.com/spreadsheets/d/1CIkz0vH-Dp3xj9K3OUvXO_mfaFwq2-qzEJbQcIyWvPg/edit

## 步驟（約 5 分鐘）

### 1. 開 Apps Script

1. 用上面連結打開試算表  
2. 選單 **擴充功能** → **Apps Script**

### 2. 貼程式

1. 刪除編輯器裡預設的 `function myFunction()...`  
2. 打開本專案檔案 `docs/google-apps-script.gs`，**全文複製**貼上  
3. 按 **儲存**（專案名稱隨意，例如 `kid-quiz-score`）

### 3. 部署成網頁

1. 右上角 **部署** → **新增部署作業**  
2. 齒輪圖示 → 類型選 **網頁應用程式**  
3. 設定：
   - **說明**：成績記錄（隨意）
   - **執行身分**：**我**
   - **具有存取權的使用者**：**任何人**（一定要選，手機網頁才能呼叫）
4. **部署** → 第一次會要你 **授權** Google 帳號 → 允許  
5. 複製畫面上的 **網頁應用程式 URL**（長得像 `https://script.google.com/macros/s/xxxxx/exec`）

### 4. 填進網站設定

編輯 `js/config.site.js`：

```javascript
SCORE_LOG_URL: "https://script.google.com/macros/s/你的ID/exec",
```

存檔後 push 到 GitHub（或只在 GitHub 網頁改這一行）。

等 GitHub Pages 更新（約 1～2 分鐘），再用手機測一次。

### 5. 確認成功

- 測驗結束畫面應顯示：**已寫入試算表「成績」工作表**  
- 試算表會多一個分頁 **`成績`**，每一列是一次測驗  

| 時間 | 小孩 | 科目 | 課次 | 模式 | 答對 | 總題 | 待確認 |
|------|------|------|------|------|------|------|--------|

### 6. 造訪記錄（v36.0 起）

網站會在背景記錄造訪（同一瀏覽器**每天最多一筆**），試算表會自動新增分頁 **`造訪`**：

| 時間 | IP | 訪客識別 | 頁面 | 版本 | 裝置 |
|------|-----|----------|------|------|------|

- **不重複 IP 數**（粗估）：`=COUNTUNIQUE(B2:B)`（B 欄為 IP）
- **更新 Script 後**：部署 → **管理部署作業** → 鉛筆 → 版本選 **新版本** → 部署（與改成績程式相同）

## 常見問題

**Q：改過 Script 要再部署嗎？**  
A：要。部署 → **管理部署作業** → 鉛筆編輯 → 版本選 **新版本** → 部署。

**Q：顯示寫入失敗？**  
A：確認部署時是「任何人」、URL 結尾是 `/exec`、Script 是綁在**同一個**試算表上建立的。

**Q：可以把網址貼給我幫忙寫進 config 嗎？**  
A：可以，把 `/exec` 那串網址傳給 Cursor，請它更新 `SCORE_LOG_URL` 並 push。
