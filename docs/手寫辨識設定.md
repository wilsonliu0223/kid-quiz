# 國語手寫辨識設定

辨識順序（都失敗 → **同音字四選一**）：

1. **筆畫辨識**（本機，hanzilookup-js）— 單字、雙字較適合  
2. **Google Cloud Vision**（雲端，經 Apps Script）— 需設定 API 金鑰  
3. **Tesseract**（本機，印刷向，準度有限）— 可關 `OCR_ENABLED: false`  
4. **同音字四選一** — 依注音從題庫挑 4 個字讓孩子點  

---

## 一、Google Vision（建議）

### 1. 建立 API 金鑰

1. 打開 [Google Cloud Console](https://console.cloud.google.com/)  
2. 建立或選專案 → **API 和服務** → **啟用 API** → 搜尋 **Cloud Vision API** → 啟用  
3. **憑證** → **建立憑證** → **API 金鑰**  
4. 建議限制金鑰只能呼叫 **Cloud Vision API**  

### 2. 填到 Apps Script

1. 打開試算表 → **擴充功能** → **Apps Script**  
2. 左側 **專案設定**（齒輪）→ **指令碼屬性**  
3. 新增：

   | 屬性 | 值 |
   |------|-----|
   | `VISION_API_KEY` | 貼上你的 API 金鑰 |

4. 把 `docs/google-apps-script.gs` 最新內容貼進編輯器（含 `handwritingVision`）  
5. **部署** → **管理部署** → 編輯 → **新版本** → 部署  

### 3. 費用

- [Vision 定價](https://cloud.google.com/vision/pricing)：Document Text Detection 每月約 **1000 次免費**  
- 家庭練習通常夠用  

### 4. 關閉雲端辨識

`js/config.site.js`：

```javascript
VISION_HANDWRITING: false,
```

---

## 二、筆畫辨識（免設定）

- 首次進入練習會背景下載字庫（約數 MB，來自 HanziLookup / Make Me a Hanzi）  
- 需網路，只需下載一次（瀏覽器快取）  
- 關閉：`HANZI_STROKE_ENABLED: false`  

---

## 三、同音四選一

- 不需額外設定，題庫愈多同音選項愈像真的同音字  
- 關閉：`HOMOPHONE_PICKER: false`（改為直接家長 PIN）
