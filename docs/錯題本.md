# 錯題本（手機本機儲存）

國語、英語**都有**，依小孩 A/B 分開存在這支手機的瀏覽器（localStorage）。

## 怎麼記錄？

| 科目 | 什麼時候記入錯題本 |
|------|-------------------|
| **國語** | 明顯寫錯、同音四選一（寫成同音字）、家長按「算錯」 |
| **英語** | **拼錯**（與標準答案不符）立刻記入；家長從「待確認」按算錯也會記 |

- **先跳過**的題不記。
- 同一題再錯會累加「錯幾次」。
- **答對**該題後會從錯題本移除（國語寫對／英語拼對）。

## 怎麼練？

1. 首頁 **錯題本** 區（有題才顯示）→ **複習國語錯題** 或 **複習英語錯題**。
2. 題數仍受首頁「本次題數」影響。
3. 考完可選 **再考本輪錯題**（只考這次錯的）。

## 英語和國語的差別

- **英語**：拼字對不對由程式比對，錯了會直接「答錯了」並記錯題本，不必每次等家長（可按「其實拼對了」請家長改判）。
- **國語**：手寫辨識可能不準，另有「我寫對了（家長確認）」。

## 家長區

長按首頁標題 → PIN → 錯題本可查看、刪除、**清空國語／清空英語**。

## 注意（手機）

| 情況 | 錯題本 |
|------|--------|
| 同一手機、同一瀏覽器 | 會保留 |
| 換手機 | 不會同步 |
| 清除網站資料 | 會消失 |
