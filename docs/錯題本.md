# 錯題本（手機本機儲存）

## 怎麼記錄？

- 存在**這支手機、這個瀏覽器**的 **localStorage**（不需 Google 帳號、不需改試算表）。
- 依**目前選的小孩（A / B）**分開存。
- 每次測驗結束，**確定答錯**的題會自動加入（含家長按「算錯」、同音四選一選錯）。
- **先跳過**的題不會記入。
- 同一個字再錯一次，會累加「錯幾次」、更新最近錯誤時間。
- **答對**該字／該英文單字時，會從錯題本**自動移除**。

## 怎麼練？

1. 首頁在「國語／英語」卡片上方會顯示 **錯題本**（有題才出現按鈕）。
2. 按 **複習國語錯題** 或 **複習英語錯題** → 只考錯題本裡的題（題數仍受首頁「本次題數」影響）。
3. 測驗結束畫面可選 **再考本輪錯題**（只考剛剛這次錯的，不必等寫入錯題本）。

## 家長區

長按首頁標題 → 輸入 PIN → **錯題本** 可查看清單、刪除單題、清空。

## 注意（手機）

| 情況 | 錯題本 |
|------|--------|
| 同一手機、同一瀏覽器（含「加入主畫面」） | 會保留 |
| 換手機 | **不會**同步，需重新累積 |
| 清除瀏覽器網站資料 / 無痕模式 | **會消失** |
| A、B 兩個小孩 | **分開**存 |

若希望兩台手機共用，日後可再考慮寫入試算表；目前以「單機練習、零設定」為主。
